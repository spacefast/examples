# Quarter Up Barcade

A demo site for a fictional '80s neon barcade in Fishtown, Philadelphia, whose hero
is a fully playable Breakout-style canvas mini-game ("Brick Buster") complete with a
live HUD, keyboard/mouse/touch controls, rising-difficulty waves, lives, a restart
flow, and a high score persisted to `localStorage`. Around the game sits real
small-business content — a grid of restored arcade cabinets, a rotating draft-beer and
cocktail list, a week of tournament events and happy-hour specials, hours and location
with an age policy, and a private-party booking form that confirms on submit. It's a
self-contained static site (`index.html` + `styles.css` + `game.js` + `ui.js`, no build
step) built mobile-first with semantic, accessible HTML and a neon/CRT aesthetic.
