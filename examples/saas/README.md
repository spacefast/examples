# Tideline

Tideline is the canonical "B2B SaaS landing page" example for the Spacefast gallery — a polished, dark-mode marketing site for a fictional issue tracker built for product and engineering teams, in the visual school of Linear, Vercel, and Height. It's a static Astro project (no SSR, just `astro build`) with a sticky nav and email signup, a hero anchored by a CSS-built product UI mock, a "trusted by" logo cloud, four alternating feature deep-dives (issues, cycles, roadmap, keyboard-driven flow), a "ship 30% faster" stat band, customer testimonials, a Free/Team/Enterprise pricing table, an FAQ accordion, and a footer CTA. The product "screenshots" are hand-built in HTML/CSS rather than stock images, so they stay crisp and fast, and all copy is specific and believable rather than placeholder text. `site/` holds the source; run `bun install && bun run build` to produce `dist/`.
