# Offcuts

A typographic home and public archive for a curated weekly newsletter, built as plain
HTML and CSS with a touch of vanilla JavaScript — no build step. "Offcuts" is written
by Mara Ellison, a Bristol designer, and the site pairs a warm paper palette and a
characterful serif (Fraunces over Newsreader) with a real reading hierarchy: a home
page that explains the letter and features the latest issue, a reverse-chronological
archive of fourteen numbered issues, three fully written issue pages (intro essay,
five hand-picked links, a "tool of the week" callout, and prev/next navigation), an
about page with author stats, and a subscribe page with plain-English fine print. The
subscribe form validates and shows a personalised success state entirely client-side,
remembering returning readers via `localStorage`. Built from `prompt.md` and published
to Spacefast — see `meta.json` for the live link.
