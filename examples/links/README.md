# Jess Park — Link in Bio

A clean, mobile-first "link in bio" page for a food creator, built as a single self-contained `index.html` with no build step. It follows the classic creator three-tier pattern — one highlighted primary call-to-action (a free meal-prep guide), a stack of tappable link buttons (latest recipe, newsletter, cookbook pre-order, shop, collabs, coaching), a featured latest-video card, and a row of social icons — wrapped in a warm editorial look (cream background, terracotta accent, Fraunces serif headings). The email signup validates and shows a friendly success state with `localStorage`, the markup is semantic and accessible, and every image is a real Unsplash food photo with descriptive alt text. See `prompt.md` to build your own version, and `site/` for the finished example.
