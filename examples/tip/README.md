# Even Split — Tip & Bill Calculator

A fast, single-purpose bill-and-tip splitter built as a self-contained static site (`index.html` + `styles.css` + `app.js`, no build step). You enter the pre-tax bill, tap a tip percentage (presets 15/18/20/25 or a custom value), keep tax in its own field so the tip is figured on the pre-tax amount — with a toggle to tip on the post-tax total instead — and set the party size with a synced stepper-and-slider. A receipt-style card recomputes on every keystroke to show the tip, tax, total, and a big per-person number, with an optional "round each person up to the next dollar" mode that surfaces the small tip cushion. A "Copy the split" button puts a tidy summary on the clipboard, all math runs in the browser with `Intl.NumberFormat`, and the markup is semantic and keyboard-accessible with a warm restaurant-receipt look (cream paper, green accent, monospace figures). See `prompt.md` to build your own version and `site/` for the finished example.
