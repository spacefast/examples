# Maya & Daniel — June 2026

An elegant multi-section wedding website for a Sonoma vineyard celebration, built as
a small static Astro project. It opens on a full-bleed hero with the couple's names
and a live countdown ticking toward the exact ceremony time, then flows through Our
Story (a dated timeline and the proposal), The Details (ceremony/reception times,
venue address with a Maps link, and dress code), Travel & Stay (airports, a hotel
room block with a booking code, and shuttle info), a photo gallery, registry cards, a
client-side **RSVP form** that validates and reveals meal/guest/song fields when a
guest is attending before showing a warm confirmation, and an FAQ accordion. Warm
ivory-and-olive palette, Cormorant Garamond over Mulish, all content driven from a
single `src/data/wedding.ts` file. Build with `bun install && bun run build`. The
copy-paste prompt that produces it lives in [`prompt.md`](./prompt.md).
