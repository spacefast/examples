# Help Fund the Maple Street Mural

A one-page fundraising campaign site for a fictional neighborhood cause — raising $24,000 to paint a 90-foot community mural — built as a no-build-step vanilla-JS app (`site/index.html` plus a CSS and JS file). It pairs an emotional cause story with a working donation flow: an animated progress bar that fills toward the goal, a donate button and modal (preset or custom amount, name, optional note, anonymous toggle) that validates input, shows a thank-you state, bumps the running total, and drops the new donor onto a live donor wall — all derived from a small config object and persisted in `localStorage`. It also has a "where the money goes" budget panel, suggested-amount tiers, a photo gallery, and a share section with a real copy-link button and pre-filled X / Facebook / email links. `prompt.md` is the copy-paste prompt that recreates the site from a few answers and publishes it live to Spacefast.
