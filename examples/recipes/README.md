# The Copper Skillet

A polished example site for Spacefast's gallery: a Mediterranean-leaning weeknight recipe blog by an invented home cook, Nora Whitfield. Built with Vite + React, it pairs an editorial cream-and-copper look with genuinely interactive cooking tools — a recipe index with live ingredient search and chip filters for course, cuisine, and dietary tags; recipe pages with a serving-size scaler that recomputes every quantity into clean kitchen fractions; checkable ingredient lists; a print stylesheet that produces a tidy single-page card; and a validating newsletter signup with a friendly success state. The full eight recipes are real, specific copy (no lorem ipsum), images come from Unsplash, and the prompt that generated it lives in `prompt.md` so anyone can rebuild a personalized version and publish it to Spacefast in seconds.
