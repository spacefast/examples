# Soon by Northpeak

A single-screen "coming soon" teaser for a fictional analytics startup, built as a
no-build-step vanilla-JS app (`site/index.html` plus a CSS and JS file). It pairs a
calm pre-dawn aesthetic — animated amber/violet/rose aurora, frosted-glass card,
fine film grain — with a confident serif headline and promise, a **live launch
countdown** that updates every second and flips to a "we're live" state when the
date passes, a **validating email capture** that swaps to a friendly "You're on the
list" confirmation and remembers returning visitors via localStorage, a waitlist
counter that ticks up once you've joined, and a row of social links. Everything runs
client-side with no backend. `prompt.md` is the copy-paste prompt that recreates the
site from a few answers and publishes it live to Spacefast.
