# Ironclad Barber Co.

A polished single-page site for a classic-meets-modern Brooklyn barbershop, built as
plain HTML, CSS, and a touch of vanilla JavaScript — no build step. It pairs a
full-bleed hero and "Book a chair" CTA with a services-and-prices grid, four named
barber profiles, a cuts gallery, and a hours-and-location block that highlights the
current day and shows a live open/closed status. The booking form validates and
returns a friendly, personalized success state entirely client-side. Built from
`prompt.md` and published to Spacefast — see `meta.json` for the live link.
