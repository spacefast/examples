# Kimchi Taco Co.

A bright, mobile-first site for a roaming Korean-Mexican taco truck whose whole job
is answering one question — *where are you right now?* The hero carries a live
"We're parked at … today" banner and the "This week's route" section reorders itself
so today comes first and today's stop is highlighted, with one-tap Google Maps
directions, an open-now / opens-at status computed from the current time, and day-off
rows shown as closed. Rounded out with a grouped menu, a catering quote form that
validates and shows a success state, a photo strip, and contact links — all
self-contained vanilla HTML/CSS/JS with no build step. Built as a Spacefast example.
