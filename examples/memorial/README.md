# Remembering Eleanor

A quiet, dignified memorial tribute page for a fictional Eleanor Margaret Hayes
(1942–2026), built as a no-build-step static site (`site/index.html`, all CSS and
JS embedded). It opens with a framed black-and-white portrait, her dates, and an
epitaph; flows into a tender life story with a drop cap, a pull-quote, and a
milestone timeline; then a photo gallery whose images open in a keyboard-navigable
lightbox with captions. Service, reception, and live-stream details sit in cards
with directions links, alongside an "in lieu of flowers" donation card. The memory
wall seeds a few heartfelt notes and accepts new ones through a validating form
that shows a gentle confirmation and persists each memory in `localStorage` so it
remains on return. `prompt.md` is the copy-paste prompt that recreates the site
from a few answers and publishes it live to Spacefast.
