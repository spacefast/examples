# Plank — Changelog

A calm, dated product changelog in the Linear / Stripe / Clerk lineage, built as a self-contained static site (`index.html` + `styles.css` + `app.js` + `feed.xml`, no build step, no framework). It tells the story of **Plank**, a fictional local-first markdown notes app: a sticky header with the product name and Subscribe actions, a hero, and a two-column release timeline (date and version on the left, content on the right) running newest-first from the v2.0 editor rewrite up to the latest daily-notes release. Each entry carries a quiet **New / Improved / Fixed** badge, real release copy with bullet details, a permalink that copies a deep link with a toast, and — for the headline releases — crisp inline-SVG mockups of the editor, calendar, search palette, and backlinks panel so the screenshots read like real product shots. A labelled filter group (`aria-pressed`) narrows the timeline by category, persisting the choice to both the URL (`?filter=new`) and `localStorage` and showing a live "showing X of Y" count; "x days ago" stamps are computed in the browser; an email subscribe form shows a friendly personalized success state with no backend; and a hand-written, valid RSS 2.0 `feed.xml` (discoverable from `<head>`) mirrors every entry. A remembered light/dark toggle, semantic HTML, keyboard support, and good contrast in both themes round it out. See `prompt.md` to build your own version and `site/` for the finished example.
