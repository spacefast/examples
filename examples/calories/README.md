# Plateful

Plateful is a single-page daily macro tracker — a browser-only food diary in the
spirit of MyFitnessPal or MacroFactor. You log meals into Breakfast, Lunch, Dinner,
and Snacks and watch an animated calorie ring and protein/carbs/fat bars fill toward
your goals, with a quick-add list of favorite foods, a date switcher for reviewing
past days, an editable goals panel, and a seven-day summary of your averages. It is
built with plain HTML, CSS, and vanilla JavaScript (no build step, no framework) and
persists everything to `localStorage`, so it works entirely client-side with no
backend. The demo ships pre-loaded with a believable high-protein diary for Maya
Okafor so the dashboard looks alive on first open. It's one of the Spacefast example
sites: copy `prompt.md` into any coding agent to generate your own personalized
version and publish it live in seconds.
