# Kaze Ramen House

A single-page site for **Kaze Ramen House**, a fictional counter-seat tonkotsu
ramen-ya in Williamsburg, Brooklyn. It opens on a full-bleed steaming bowl, then
walks through a real, priced menu (ramen, gyoza and small plates, rice bowls and
drinks), the chef's 18-hour-broth story, a tap-to-enlarge photo gallery, and a
hours-and-location block with a live "open now / closed now" badge and an embedded
map. Built as plain HTML & CSS with a small vanilla-JS enhancement (sticky header,
mobile nav, scroll reveals, lightbox, opening-hours logic) — no build step, fully
responsive and accessible. It's both a polished example and the output of the
copy-paste prompt in `prompt.md`.
