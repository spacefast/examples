# RamHacks 2026

An energetic, dark-mode landing site for a free 36-hour student hackathon at the
fictional Rampart State University, built to answer the two questions every prospective
hacker has — *when does it start* and *why should I come?* The hero carries a real
JavaScript countdown that ticks down to kickoff second-by-second and gracefully flips
to a "Hacking is live!" state during the event and a "that's a wrap" state after it,
while the schedule section toggles between a Saturday and Sunday agenda built from a
data array (opening ceremony, beginner workshops, midnight snack, judging, closing).
Rounded out with a grand-prize block plus seven sponsor-backed tracks, tiered sponsor
logos with a "Sponsor us" CTA, an accessible keyboard-operable FAQ accordion, a
validating registration form with a success state, and an MLH Member Event badge —
all self-contained vanilla HTML/CSS/JS with no build step. Built as a Spacefast example.
