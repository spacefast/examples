# Prism Breaker

Prism Breaker is a fully playable neon-synthwave brick-breaker arcade game built as a self-contained HTML5 canvas site — no build step, no asset files. You bounce a glowing ball off a paddle to shatter colored brick walls across five hand-designed levels (The Wall, Pyramid, Lattice, Fortress, and The Prism), each ramping up the ball speed, while catching falling power-ups — multi-ball, wide paddle, slow-mo, laser, and the rare extra life — and chasing a best score that persists in `localStorage`. It runs at 60fps with real wall/paddle/brick collision physics, particle bursts, ball trails, screen shake, and Web Audio synth blips, and it plays with keyboard, mouse, or touch (on-screen controls appear on mobile). It's the canonical Spacefast "Tools, apps & games" example: a real game loop and state machine that's instantly verifiable by whether it's actually fun to play.
