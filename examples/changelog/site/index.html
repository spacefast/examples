What's new in Plank

Plank is a calm, local-first markdown notes app for people who think in plain text. Here's everything we've shipped — features, improvements, and fixes — newest first.

Jun 24, 2026 v2.4.0
New

Daily notes & a calendar in the sidebar

Open a fresh note for any day with one keystroke. Press ⌘D and Plank creates (or jumps to) today's note from your template — perfect for journaling, standups, and a running log of what you actually did.

  • A month calendar lives in the sidebar; dots mark days that already have a note.
  • Set a Daily Note template with placeholders like {{date:dddd}} and {{weather}}.
  • Yesterday / tomorrow navigation with ⌘[ and ⌘].
Shipped by Mara & Devin
Jun 12, 2026 v2.3.2
Fixed

Sync conflicts on large vaults, resolved

A handful of you with 5,000+ note vaults hit phantom conflicts when two devices saved within the same second. We were comparing modified-times at second resolution; Plank now uses a content hash plus millisecond timestamps, so a clean save never looks like a conflict.

  • Fixed duplicate note (conflicted copy) files appearing after a quick edit-and-switch.
  • The sync indicator no longer gets stuck spinning when a folder is renamed mid-sync.
  • Reduced memory use during the initial index of very large vaults by about 40%.
May 28, 2026 v2.3.0
Improved

Full-text search that keeps up with your typing

We rebuilt search on a new on-disk index. Querying a 10,000-note vault now returns results in under 15 ms — fast enough that results stream in as you type, with the matched phrase highlighted in context.

  • New operators: tag:#idea, path:journal/, and "exact phrase".
  • Recent searches and saved searches now live at the top of the palette.
  • Search ranks recently-opened notes higher, so the thing you want is usually first.
May 9, 2026 v2.2.0
New

Wikilinks and a live backlinks panel

Type [[ to link to any note — Plank autocompletes titles as you go, and creates the note if it doesn't exist yet. Every note now has a backlinks panel showing where else it's mentioned, so your vault quietly turns into a connected graph.

  • Hover a wikilink to preview the linked note without leaving the page.
  • Rename a note and every [[link]] to it updates automatically.
  • An optional local graph view shows a note and its immediate neighbours.
Shipped by the editor team
Apr 22, 2026 v2.1.1
Fixed

Pasted code blocks keep their indentation

Pasting a Python or YAML snippet into a fenced code block was helpfully — and infuriatingly — collapsing leading whitespace. Plank now preserves indentation exactly inside ``` fences, and won't auto-format what's between them.

  • Tab inside a code block inserts a tab (or spaces, if that's your setting) instead of moving focus.
  • Smart-quote substitution is disabled inside code and inline `code`.
  • Fixed the cursor jumping to the top of the note after pasting a very long block.
Apr 3, 2026 v2.1.0
Improved

A faster, smarter command palette

The command palette (⌘K) is now fuzzy, ranked, and fast. It learns the commands you reach for and floats them to the top, and you can run almost anything from it — create a note, toggle a setting, jump to a heading, or insert a template.

  • Type a few letters of any command; matches highlight the characters you typed.
  • New quick actions: Move note…, Pin to sidebar, and Export to PDF.
  • Palette opens in roughly half the time it used to on first launch.
Mar 14, 2026 v2.0.0
New

Plank 2.0 — a brand-new editor core

This is the big one. We rewrote Plank's editor from the ground up for a live-preview experience: markdown styles itself as you type, with no separate preview pane to toggle. Bold looks bold, headings look like headings, and the raw syntax appears only on the line your cursor is on.

  • Native tables, callouts, task lists, and math ($…$) rendered inline.
  • Everything is local-first — your notes are plain .md files on disk, yours to keep.
  • A new theming engine with light, dark, and a warm "Paper" theme out of the box.
