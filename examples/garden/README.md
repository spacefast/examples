# Maple Street Community Garden

A single-page site for **Maple Street Community Garden**, a fictional volunteer-run,
certified-organic neighborhood garden with 48 rentable raised beds in Rosewood. It
opens on a sunny full-bleed beds photo, then walks through the garden's origin story
and stats, a four-card plots-and-pricing grid (Starter, Standard, Family, and a
waist-high accessible bed) with an "every plot includes" list, a 3-step **waitlist
signup form** that validates, shows a friendly success state, and remembers the
visitor via `localStorage`, a volunteer-workdays block with a live "next 2nd-Saturday
workday" badge computed in the browser, six garden-rule cards, a dated seasonal events
list, a tap-to-enlarge seasonal photo gallery, and a find-us block with gate-access
notes and an embedded map. Built as plain HTML & CSS with a small vanilla-JS
enhancement (sticky header, mobile nav, scroll reveals, lightbox, next-workday and
form logic) — no build step, fully responsive and accessible. It's both a polished
example and the output of the copy-paste prompt in `prompt.md`.
