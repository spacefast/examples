# Milestone — savings-goal calculator

Milestone is a self-contained, vanilla-JavaScript savings-goal calculator: you enter a target amount, a deadline, what you've already saved, and an expected annual return, and it solves the one number that matters — the monthly contribution you need to set aside — alongside a per-week equivalent, a compound-growth area chart climbing toward the goal line, a breakdown of contributions versus interest, a what-if explorer whose sliders move the projected goal date in real time, and a year-by-year milestone table. The compound-interest math is exact (end-of-month contributions, monthly compounding) and inverted for the "when would I hit it" scenarios, with edge cases handled for 0% cash, an already-met goal, and interest-alone growth. Everything runs in the browser with no backend, inputs persist to `localStorage`, and the site ships as the Spacefast example `milestone-savings-calculator`.
