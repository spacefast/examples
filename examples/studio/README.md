# Northbound Studio

A work-first portfolio for a fictional brand-and-web design studio in Portland, Oregon,
built as a no-build vanilla-JS site (`index.html` + `styles.css` + `app.js`). It opens
on a confident statement hero ("We build brands people remember"), then a **selected
work** grid of six projects — *Aster &amp; Oak*, *Harbor Collective*, *Cadence*,
*Northstar Roasters*, *Veridian*, and *Juniper Press* — with **Branding / Web / Motion**
filter chips toggled in JavaScript. A dark featured case study includes a tap-to-compare
before/after, followed by a numbered services list, a studio-philosophy + team section, a
client logo strip, and an inline-validated contact form that shows a personalized success
state. Editorial styling pairs a Fraunces serif display with Space Grotesk, on warm paper
with a vermilion accent; it's mobile-first, accessible (skip link, labelled fields,
keyboard support), and ships with the Spacefast badge.
